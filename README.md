# TCC Mapa Paulista

## Instruções para iniciar o servidor backend

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor backend:
   ```bash
   node src/server.js
   ```

3. Inicie o servidor frontend:
   ```bash
   npm run dev
   ```

4. Acesse a aplicação em [http://localhost:5173](http://localhost:5173)
